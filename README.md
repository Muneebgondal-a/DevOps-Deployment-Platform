# DevOps Control Center

A Flask-based DevOps dashboard that helps monitor system resources, manage Docker containers, and automate deployments using Jenkins.

## Features

- System Monitoring
- Docker Integration
- Docker Container Management
- Git Integration
- Jenkins CI/CD Pipeline
- Flask Web Dashboard

## Technologies Used

- Python
- Flask
- Docker
- Git
- GitHub
- Jenkins
